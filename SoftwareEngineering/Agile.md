# Agile이란?
  - 개발해나가면서 피드백을 통해 필요한것들을 바탕으로 주도적으로 개발
  - 짧은 주기의 개발을 반복하여 큰 프로젝트를 완성
  - 빠른 피드백과 협력이 중요
  - agile <->Water fall
  ```
  Agile 선언문
  * 공정과 도구보다 개인과 상호작용을
  * 포괄적인 문서보다 작동하는 소프트웨어를
  * 계약 협상보다 고객과의 협력을
  * 계획을 따르기보다 변화에 대응하기를
  ```
