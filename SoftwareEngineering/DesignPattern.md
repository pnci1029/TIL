    효율적인 코드 작성을 위한 방법론
    
    객체지향 프로그래밍 설계 시 자주 발생하는 문제를 피하기 위해 사용되는 패턴

    여러 사람이 협업하여 개발 시 다른 사람이 작성한 코드나 기존 코드를 수정하거나
    추가 개발 시 버그나 성능저하가 발생하여 추가적인 리소스가 발생함

    특정 규약을 만들어서 정리한 것.
