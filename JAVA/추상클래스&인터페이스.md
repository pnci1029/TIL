# 추상클래스
  - 하위클래스들의 공통점을 모아 추상화하여 만든 클래스
  - 필드, 메서드 생성자 생성 가능 -> 상속받는 클래스들의 중복 값들 통합 및 확장
  - 클래스 내 추상 메소드가 1개 이상 포함된 클래스(<->일반 클래스)
  - 단일상속만 허용
  - 상속을 위한 클래스
  - 인스턴스화 불가능

# 인터페이스
  - 모든 메소드가 추상메소드로 구성된 경우
  - 껍데기 함수를 만들고 상속받은 클래스에서 인터페이스의 구현을 강제화
  - 타 클래스에서 다중 상속이 가능
  - 인스턴스화 불가능
